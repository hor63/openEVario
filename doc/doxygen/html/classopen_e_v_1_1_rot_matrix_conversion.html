Detailed Description

The algorithm to construct a rotation matrix from two vectors comes from: http://stackoverflow.com/questions/23166898/efficient-way-to-calculate-a-3x3-rotation-matrix-from-the-rotation-defined-by-tw

The algorithm to decompose a rotation matrix into the Euler angles comes from http://nghiaho.com/?page_id=846
