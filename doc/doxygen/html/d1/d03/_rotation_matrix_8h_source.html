33 /***
34  * \class RotationMatrix
35  * \brief Rotation matrix to transform the global coordinate system into the plane's coordinate system.
36  * Rotation matrix to transform the global coordinate system into the plane's coordinate system
37  * References are \ref https://en.wikipedia.org/wiki/Euler_angles#Vehicles_and_moving_frames. The matrix used is Tait-Bryan angles from
38  * \ref https://en.wikipedia.org/wiki/Euler_angles#Rotation_matrix.
39  * A IMHO more concise description is in German \ref https://de.wikipedia.org/wiki/Eulersche_Winkel#Drehfolgen_in_der_Fahrzeugtechnik
40  * I am using the Y-Y-Z order as defined in DIN 9300 for aircraft rotational order.
41  * I am also using the coordinate system definition according to DIN 9300:
42  * - x = horizontal North.
43  * - y = horizontal East
44  * - z = perpendicular *down*; moving up decreases z, but on the other hand earth gravity is +9.81m/s^2 in z direction.
45  * I am using the common names of the angles. In the description you find the greek designations from DIN 9300 (see the German Wikipedia)
46  */
